# 任务

1. 纯html+css+js实现，不依赖任何框架
2. 实现一个调整图片纹理的工具，用户上传图片后，先计算图片中所有的像素点的色值，然后把色值按照“欧氏距离聚类”算法进行聚类，最后根据聚类结果，调整图片中所有的像素点的色值，使图片的纹理更加统一
3. 聚类的个数可以让用户自己调节，默认20个，最多200个
4. 左侧是原始图片，右侧是调节后的图片
5. 图片要保持原始图片的大小，不能拉伸或压缩
6. 做一个可选的加躁点的功能，用户可以在调整后的图片上添加一些噪点，使图片更加真实
7. 躁点功能就是对每个像素点的RGB通道进行10以内的加减法，10这个数值也可以让用户自定义，最高50
